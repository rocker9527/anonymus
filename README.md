## Anonymus Bot

Telegram bot to send messages anonymously (not forward) to your group.

## Tutorial

  - Create config.py

    ~~~
    cp config.sample.py config.py
    ~~~

  - Update Config

    ~~~
    "BOT_TOKEN": "your_token",
    "BOT_MANAGER": your_user_id,
    ~~~

  - Add bot to your group

    ~~~
    /setgroup
    ~~~

  - Send text or media to Bot

  - Enjoy
